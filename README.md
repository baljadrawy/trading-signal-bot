# 🤖 Trading Signal Bot

بوت تداول ذكي يراقب العملات المشفرة على Binance، يحلّلها تقنياً، ويرسل إشارات عبر Telegram مع نظام موافقة شرعية.

## ✨ المميزات

- **مسح تلقائي** للعملات كل 5 دقائق
- **تحليل فني متكامل**: RSI, MACD, EMA, Bollinger, Volume, Order Book
- **مراجعة Claude AI** لكل إشارة قبل الإرسال
- **نظام Whitelist شرعي**: موافقتك مطلوبة للعملات الجديدة
- **إدارة المخاطر**: وقف تلقائي عند 3 خسائر متتالية
- **تعلم ذاتي يومي**: يحسّن أوزان المؤشرات بناءً على نتائج الصفقات
- **Paper Trading**: للاختبار بدون مال حقيقي

## 🏗️ المكونات

| Container | الوظيفة |
|-----------|---------|
| `scanner` | يمسح العملات ويجمع بيانات السوق |
| `analyzer` | يحلّل المؤشرات الفنية |
| `signal_engine` | يختار أفضل الإشارات |
| `claude_review` | يراجع الإشارات بـ Claude AI |
| `telegram` | يرسل الإشارات ويستقبل أوامر التحكم |
| `postgres` | قاعدة البيانات |

## 🚀 التشغيل السريع

```bash
git clone https://github.com/baljadrawy/trading-signal-bot.git
cd trading-signal-bot
cp .env.example .env
nano .env        # أضف مفاتيحك
./setup.sh       # شغّل النظام
```

## 📋 أوامر Telegram

| الأمر | الوظيفة |
|-------|---------|
| `/status` | حالة النظام اليومية |
| `/whitelist` | العملات المعتمدة شرعياً |
| `/stats` | إحصائيات آخر 7 أيام |
| `/pause` | إيقاف مؤقت |
| `/resume` | استئناف |
